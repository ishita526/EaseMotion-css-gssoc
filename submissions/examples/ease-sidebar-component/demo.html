All Variants Reference

Quick reference cards for every sidebar variant and its CSS class.

Default

Default Sidebar

Clean base sidebar with nav links, icons, badges, profile section, and active state indicators.

ease-sb--collapsed

Collapsible Sidebar

Smooth width transition to icon-only mode. Labels and badges fade out gracefully.

ease-sb--icon-only

Icon-Only Sidebar

Permanently compact with tooltip hints on hover via data-tooltip attribute.

ease-sb--dark

Dark Sidebar

Deep dark background with high-contrast text and muted icon colors.

ease-sb--light

Light Sidebar

Explicit clean white theme override with subtle borders and hover states.

ease-sb--gradient

Gradient Sidebar

Rich purple-to-deep gradient background for creative applications.

ease-sb--glass

Glassmorphism Sidebar

Frosted glass effect with backdrop-filter blur and semi-transparent layers.

ease-sb--fixed

Fixed Sidebar

Position: fixed — stays pinned to the left viewport edge on scroll.

ease-sb--floating

Floating Sidebar

Elevated with margin, border-radius, and dramatic floating shadow.

ease-sb--offcanvas

Mobile Off-Canvas

Slides in from left with a blurred overlay backdrop. Toggle via JS class.
